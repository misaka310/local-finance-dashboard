# AGENTS.md

このリポジトリをCodexなどのエージェントで編集する場合の固定ルールです。

## 最優先

- Gmail OAuthクライアントJSON、OAuthトークン、SQLite DB、実明細データをGit管理に入れない。
- `secrets/` と `data/` の実ファイルは読まない・表示しない・コミットしない。
- Gmail権限は `gmail.readonly` のままにする。送信・変更権限へ広げない。
- 新規パッケージ追加、外部スクリプト実行、外部バイナリ実行は実行前に理由を説明する。
- PayPayカードや銀行サイトへのログイン自動化は追加しない。

## 実装方針

- UIは `frontend/` の静的ファイルで完結させる。
- ローカルサーバーはPython標準ライブラリのHTTPサーバーを使う。
- 保存先はSQLite。
- カテゴリは大項目固定・中項目追加可能の前提で扱う。
- 支払い元とカテゴリを混同しない。

## セキュリティ方針

- メール本文全文はDBに保存しない。
- 保存するのは、日付、店名、金額、支払い元、カテゴリ、Gmail message id、取り込み日時など必要最小限にする。
- OAuthトークンはkeyring経由でOSの資格情報保管領域に保存する。
