# 運用メモ

## 基本コマンド
```powershell
.\scripts\03_sync_now.ps1
.\scripts\04_run_app.ps1
```

`.\scripts\04_run_app.ps1` 起動後、`http://127.0.0.1:<port>/` または `/index.html` でUIを開きます。

## 月/年表示の使い方
- 画面上部の `月` / `年` ボタンで表示単位を切り替えます。
- 月表示は `YYYY-MM` の明細・集計、年表示は `YYYY` の年間明細・年間集計を表示します。
- 支出/収入タブ切替や中項目表示ON/OFFを行っても、現在の月/年モードは維持されます。

## 期間移動と「最新へ」
- 月表示では前月/次月、年表示では前年/翌年に移動できます。
- `最新へ` ボタンは、
  - 月表示: データが存在する最新月へ移動
  - 年表示: データが存在する最新年へ移動
- すでに最新期間にいる場合は `最新へ` は無効化されます。

## データ範囲外への移動制限
- `transactions` テーブル上の最小/最大の月・年をもとに、移動可能範囲を制限します。
- データが存在しない期間へは移動できません（前へ/次へボタンが無効化）。

## Gmail同期ボタン
- `Gmail同期` ボタンでGmailの取り込みを実行します。
- 同期中の表示は `Gmail同期中…` になります。

## 年表示で見る内容
- 年表示では年間の `収入` `支出` `残高` を表示します。
- 円グラフと凡例は、その年の選択中タブ（支出/収入）のカテゴリ比率を表示します。

## PayPay CSV 取り込み
```powershell
.\scripts\07_import_paypay_csv.ps1 "%USERPROFILE%\Downloads\detail*.csv"
```

## セキュリティとGit管理
- `data/` と `secrets/` の実ファイルはコミットしません。
- `.env` / OAuthトークン / 認証情報はコミットしません。
- `data/classification/category_rules.example.csv` のようなサンプルのみ共有します。
